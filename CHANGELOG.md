# Changelog

## 0.1.0

Initial standalone ArcHub CMS release.

- Extracted ArcHub source modules into `src/archub_cms`.
- Added standalone FastAPI app factory and demo content seeding.
- Added package-owned templates and static assets.
- Added product integration ports and environment-backed settings.
- Added standalone RAG corpus registry and external-indexer hook.
- Added GitHub-ready CI, docs workflow, release notes and demo site.
