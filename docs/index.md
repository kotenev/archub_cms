# ArcHub CMS

ArcHub CMS is a standalone headless CMS and Content Builder for FastAPI hosts.
It was extracted from a production bot backoffice and productized as an
Apache-2.0 package.

## What is included

- Umbraco-style document types, reusable data types and templates.
- Hierarchical content tree with draft, publish, unpublish and trash states.
- Version history, preview tokens, redirects, domains, cultures and segments.
- Content Builder block registry with strict normalization and HTML rendering.
- Published delivery API for content, tree, search, tags, RSS and sitemap.
- Runtime exports for bot resources and RAG materials.
- Standalone demo app and GitHub Pages documentation.

## Quick links

- Backoffice route: `/admin/archub`
- Published site: `/cms`
- API docs: `/api/docs`
